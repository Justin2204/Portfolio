# A2 - Security Concepts Used in Public Space

## Overview
This activity examines security concepts used in a public space. The aim is to identify practical security measures, explain how they are applied, and reflect on how they help protect people, property, systems, and information.

## Public space investigated
- School library

## Security concepts identified
- Access control
- Surveillance and monitoring
- Asset protection
- Acceptable use and policy enforcement

## Description and purpose

### 1. Access control
- **Security concept:** Access control
- **Description:**  
  The library is a public-facing space, but access to some areas, systems, or services may still be restricted. For example, students may need a school ID or login credentials to borrow items, use computers, print documents, or access online library resources. Staff-only rooms, storage areas, or service desks may also be restricted.
- **Purpose:**  
  To ensure only authorised users can access certain resources, systems, or restricted areas.
- **How it improves security:**  
  This reduces unauthorised access to library systems, protects resources, and helps manage who can use services and facilities.

### 2. Surveillance and monitoring
- **Security concept:** Surveillance and monitoring
- **Description:**  
  Libraries often use visible security measures such as CCTV cameras, staff supervision, and service desk oversight to observe activity in the space. This helps staff monitor behaviour, respond to incidents, and identify misuse or theft.
- **Purpose:**  
  To deter inappropriate behaviour and help detect incidents when they occur.
- **How it improves security:**  
  Surveillance improves safety for students and staff, discourages theft or damage, and provides evidence if an incident happens.

### 3. Asset protection
- **Security concept:** Asset protection
- **Description:**  
  Libraries contain valuable assets such as books, computers, printers, furniture, and study equipment. Security measures may include barcode systems, borrowing records, device supervision, and physical placement of equipment where staff can observe it.
- **Purpose:**  
  To protect library resources from theft, loss, or damage.
- **How it improves security:**  
  These measures help the library track resources, reduce loss, and ensure materials remain available for users.

### 4. Acceptable use and policy enforcement
- **Security concept:** Policy enforcement
- **Description:**  
  Libraries usually operate under rules about computer use, internet access, borrowing, noise, behaviour, and respectful use of shared spaces. These rules may be displayed through signs, student policies, or verbal reminders from staff.
- **Purpose:**  
  To create a safe, orderly, and secure environment for all users.
- **How it improves security:**  
  Clear rules reduce misuse of systems and facilities, support safe behaviour, and help staff respond consistently when issues arise.
