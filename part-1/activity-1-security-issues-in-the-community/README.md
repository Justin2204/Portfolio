# Activity 1 - Security Issues in the Community

## Overview
This folder contains the supporting material for Activity 1 in Part 1 of my cybersecurity portfolio.

## Activity description
This activity focuses on identifying cybersecurity issues in community organisations, explaining their impact, and recommending ways to improve their security posture.

## What I did
- Selected community organisations to investigate
- Identified possible cybersecurity issues
- Considered the risks and impacts of those issues
- Developed recommendations to improve security
- Collected supporting evidence and notes

## Evidence included
The supporting materials for this activity may include:
- screenshots
- notes
- reports
- supporting documents
- reflections on what was learned

## Outcome
This activity helped me develop practical skills in identifying real-world cybersecurity risks and proposing suitable mitigations.

- `evidence/` contains screenshots, PDFs, and other proof
- `files/` contains related notes, scripts, reports, or working files
